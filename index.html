Install the connector

A connector is Claude's version of an integration. Add it once and the skills show up in every conversation, with new ones appearing automatically as I ship them.

  1. Download Claude Desktop if you haven't already.
  2. Open Settings (click your name, bottom-left), then Connectors.
  3. Click Add custom connector.
  4. Name it Jen's Skills and paste this URL:
    https://your-skills-url.example.com
  5. Click Add, toggle it on, and start a new conversation.
  6. Optional: open the connector and set Tool permissions to Always allow, so Claude doesn't ask before each skill.
Can't install the connector? Grab the skills pack on GitHub, it works with any AI tool.

Get started

  1. Open Claude and create a new project. Cowork is preferred if you have it (top-left, next to Chat). Chat works too.
  2. Start a conversation in that project and type: Build my profile
  3. Setup scans the tools you've connected, then asks a few questions to fill the gaps. About 5 minutes.
  4. When it's done, copy the project instructions it generates, open Instructions (top right, pencil icon), and paste them in.

Your first 10 minutes

Don't stop at the install, run one workflow right now. The daily brief is the best first test. Curious what else is in there? Type help in any conversation to see every skill and its trigger phrase. If something isn't working, disconnect and reconnect the connector, or restart Claude. That fixes most things.

Common questions

Is my data safe?

The skills are instructions for Claude, not a pipe to me. Your tools stay connected through Claude's own connectors, and I only see that the skills are being used, not what's inside them. Check with your company that your Claude account is set up so Anthropic doesn't train on your data.

Will it make things up?

It works from what's in the tools you connect: your calendar, email, CRM, and meeting recordings. AI can still get things mixed up, so treat yourself as the human reviewer on anything it drafts, especially before it goes to someone else.

Does it work with my tools?

Any tool with a Claude connector: your meeting recorder, CRM, email, and calendar. You can check yours in Claude's connector directory. If yours isn't listed, the workflows still run, you'll just paste context in manually.

What's the catch?

No catch. It's free, no signup or email gate, and you can remove it anytime. I built this because it makes my own work better, and I figured it might do the same for you.

How do I uninstall it?

Settings, then Connectors, then remove it. That's it. Nothing else to clean up.
